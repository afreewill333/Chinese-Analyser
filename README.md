而因為更深入的研究，於是有更多朋友和我推薦相關的其他服務，也就這樣源源不絕的又發現了不少好工具。

今天要推薦的，就是另外一個「線上檢查英文文法」的服務：「 Grammarly 」，他有兩大特色，第一大特色是他的免費版可以檢查出 100 種英文文法錯誤，以及包含其他的拼字錯誤，而他的付費版則號稱可以檢查出 250 種以上的語法錯誤。

第二大特色則是「 Grammarly 」的介面非常清爽，閱讀與書寫起來很舒適，速度又快，甚至可以當作一個「線上英文寫作軟體」來使用！

而且「 Grammarly 」和前幾天介紹的 Ginger 還有一個很大的不同，就是「 Grammarly 」提供的文法錯誤解釋更為詳盡。




我推薦使用「 Grammarly 」的網頁版，登入帳戶後，可以把文章上傳或剪貼到其線上文書工具中，就可以自動幫我們檢查英文文章的各種錯誤，包含了：

1.找出標點錯誤
2.找出拼寫錯誤
3.找出語法錯誤
4.找出時態錯誤





（根據網友測試回報，需要使用 chrome 才會看到免費方案）在「 Grammarly 」上註冊一個新帳號，依序完成到最後一步時，會有兩種方案可以選擇，選擇左邊的方案就能免費使用。





而且「 Grammarly 」的書寫介面非常賞心悅目，我建議直接就把「 Grammarly 」當作寫作軟體來使用，在上面一邊寫你的英文信、英文報告，一邊讓「 Grammarly 」幫你即時檢查出各種文法錯誤。

這時候錯誤的文章都會顯示在右方。





「 Grammarly 」另外一個優點就是他的錯誤解釋很詳細，一個時態錯誤也會有詳細的原因解釋，甚至告訴你幾種可能的變化型態，讓你不只找出錯誤，更能學會正確的使用方法。

可以說「 Grammarly 」是能夠方便你進行英文寫作，又能進行文法即時檢查糾正的利器，很值得需要強化英文寫作技巧的朋友試試看。
