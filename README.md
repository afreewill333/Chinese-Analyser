Grammarly：我的私人语法老师
目录 Mac App2016-03-15

经常需要英文写作的 Mac 粉儿肯定都听说过 1Checker 这类的英文语法检查纠错软件，但在 MAS 上此类软件玩票性质的太多，真正能从实效上帮助用户提高英文写作水平的工具在 Mac 平台真的不太多，今天我们介绍一款跨平台的英文写作纠错工具：Grammarly，来看看他的强大之处都有哪些。

Grammarly 可在 PC/Mac/Web/Office 套件（仅支持 PC 版）上使用， 在 Web 上配合 Safari/Chrome 插件，你可以在网页的文字编辑中随时调用 Grammarly 进行词句应用检查，在 Office 套件的 Word、Outlook 平台上可享用丰富的语法纠错服务和功能，下面我们先主要以 Mac 客户端为环境介绍下 Grammarly 的基础功能和高级功能。

Grammarly Mac 客户端完全以 Web 版为原型打造（封装），你可以直接将文章全文拷贝进入 Grammarly，或是直接使用 Import 将文章文件导入进来。

Grammarly 提供拼写校正（Spell Checking），语境分析下的词汇应用纠正（Contextual Spelling Check），语法规则纠正（Grammar），标点符号纠正（Punctuation），句式架构纠正（Sentence Structure）等基础功能，在高级功能中，你还可以享用文章类型预判（Document Type，可根据不同类型的文章进行校对），文章原创性检查（Plagiarism），单词推荐（Vocabulary enhancement）以及文章校对服务（Proofreading，邀请专业人士为你的文章找问题，需要另花钱）。





贴入（导入）文章后，Grammarly 会自动对整个文章进行检查，你可以在窗口底部看到一个信息实时统计栏，这里显示着 Document Type，字数，问题数量，以及 Grammarly 根据上文几个校对角度进行检查后给出的综合评分，帮助用户改进文章。

Grammar 检查


检查结束后，所有的找出的问题和建议都会显示在文章右侧，有问题的文字或段落会有绿线标记在下方，比如一个段落整个被绿线标记出来，这说明整个段落的编写有问题，单击这个段落任意位置可调出右侧的修正建议，原来是 叙述冗长方面的（Wordiness），Grammarly 会建议你将长句切断为几个短句，方便读者理解。在修正建议窗口左下角有一个 Expand 按钮，点击后可查看更加详细的改进解释和建议。



这里是一个比较典型的语法应用错误，软件会给出“修改参考”以及 Expand 下更详细的解释，同时还会推荐两个正确与错误的应用案例帮你加深印象。最后，单击一下“修改参考”即可自动将错误修正（可选择 Undo 回撤操作）。



Grammarly 提供实时显示的同义词（Synonym），当你双击任意一个单词时软件会给你一个同义词推荐作为参考，对于经常需要改写文章的用户来说，这个功能绝对必要。



对于在文中反复出现的单词，软件会提示你想想用其他词汇替换一下，要不会显得异常啰嗦，特别是对于那些要发布到网络上的文字，如果是关键词出现次数过多会影响文章的优化排名。



文章原创性检查（Plagiarism）
Grammarly 通过后台自建的庞大网页数据库检索技术，可为你的文章原创性做检查，我觉得只要是你自己一个字一个字敲出来的，那么原创性是没有必要操心的，但对于需要做英文伪原创的用户来说，Plagiarism 就很有必要了，经过深入编辑，你需要看看成文与网络上的原创文或其他伪原创文在内容上的重复度（Unoriginal）是多少，如果高于20%，还是建议你重新写吧。



使用 Proofreading 打造精品文章
开发者在高级功能的基础上还提供了更深层次的服务项目，用户可在开发者的渠道中邀请英文文字专家为文章把关，这个服务是 7×24 小时的，你可以随时上线找专家帮你改文章，付费标准有三种，分别是 24 小时交稿（$15.66/每 783 个字）,3小时交稿（$31.32/每 783 个字）以及30分钟交稿（$62.64/每 783 个字），服务定价扣得很细，当然也是我用过的最贵的，但可以看出 proofreader 水平应该不错，感兴趣的朋友可以试试。

配合 Office Word/Outlook 使用
Grammarly 为 PC 用户单独开发了 Office 内置工作程序（下载），可以在 Word/Outlook 内使用语法纠正等任务，就不用你再去通过 Web 工作了，安装好内置版 Grammarly 后，启动 Word 或 Outlook，你会在视图频道的右侧（Word）看到 Grammarly 工作区，里面的操作方式是和 Web 版一样的。



总的来说，Grammarly 功能强大，既能为初级用户提供词语拼写检查，同义词推荐等 5 大语法检查，又能为专业用户提供文章原创性检查（Plagiarism）、Proofreading 等服务，如果订阅了高级版（Premium Plan，支持 7 天无理由退款），你还能享受更多专业的语法词汇校正参考（比基础版多出 150 多种），更多的高级语法指导，单词推荐等功能，高级版的付费分三种，第一种是月付，每月价格为 $29.95，第二种是按季度付费，每月价格为 $19.98，第三种就是年付，每月价格为$11.66。



Grammarly 适合使用人群非常广泛，比如像经常从事英文编写、创作或是正在准备雅思、托福准备出国，亦或是在国内生活工作的朋友都可以把他当做提高英文写作能力的助手。
